# Player Management System ⚽

##  Overview

Player Management System is a web-based application designed to manage football players, teams, and leagues efficiently. The system allows users to perform operations such as adding players, managing teams, and tracking performance data.

## Features

* Add, update, and delete players
* Manage teams and leagues
* Store player details (age, position, rating, etc.)
* Web interface for interaction
* Database integration for data storage

## 🛠️ Technologies Used

* Java
* HTML/CSS
* Simple Web Server
* Database (H2 / SQL-based)

## 📂 Project Structure

* `controller/` → Handles API and routing
* `service/` → Business logic
* `dao/` → Database access
* `model/` → Data models
* `resources/` → HTML files

##  How to Run project

1. Clone the repository:

   ```
   git clone https://github.com/TORRIC67/playermgs.git
   ```
2. Open the project in your IDE (VS Code / IntelliJ)
3. Run the `Main.java` file
4. Open your browser and access the system
